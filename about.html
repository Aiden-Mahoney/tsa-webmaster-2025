Purpose

The Gettysburg Community Resource Hub is designed to help residents quickly find local support services, nonprofit organizations, volunteer opportunities, and community programs in the Gettysburg area.

How to Use

Visit the Resource Directory to search and filter resources by category or access type. Use the Home page quick-access buttons to jump straight to common needs.

Suggestion Box

If you know a resource that should be included, use the “Submit a Resource” page to share it.
