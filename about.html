Note to Judges

Navigation: Use the menu at the top of each page to navigate the site.

Quick Access: Home page buttons jump directly to filtered results in the Resource Directory.

Directory: Resources can be searched and filtered by category and access type.

Submission: The resource submission form is fully functional and displays a success message.

TSA Background

This is my 4th year participating in TSA. I selected the Webmaster event to step outside of my comfort zone and to learn new skills related to website design, structure, and user experience.

How This Website Meets the Theme Requirements

  • Interactive community resource directory with search and filtering
  • Highlighted community organizations
  • Submission form for users to suggest new resources
  • Additional content to improve usability and accessibility
